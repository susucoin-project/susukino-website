--------
言論の自由の保護

概要
--------
Susucoin は自由な言論を保管しておくために、トランザクション上に格納容量を追加して設計された PoW の暗号通貨です。
Susucoin は独自のジェネシスブロックを持ち、Bitcoin Core プロジェクトを元にしたウォレットソフトウェア上に独立したブロックチェーンを持っています。
Susucoin Core チームは複数のフルノードを管理し、 Susucoin Core プロジェクトは GitHub リポジトリ上での開発を通して維持されます。
更新された Bitcoin Core のセキュリティパッチは自動的に Susucoin プロジェクトにも適用され、Bitcoin Coreから追加される新機能は必要に応じて Susucoin プロジェクトにも適用されます。
Susucoin Core チームもまた、Susucoin 独自の新機能を開発し、追加していく予定です。
我々はすでに言論の自由を守る多くの匿名プラットフォームと、そのコミュニティの中で Susucoin を使用する API や機能を統合する第一者提携を結んでいます。

技術
--------
Susucoin は、SHA256 アルゴリズムを使用しており、2分間隔を目標としてブロック毎に採掘難易度の再キャリブレーションを行います。
採掘難易度キャリブレーションのアルゴリズムは、1～60ブロックの間はDigishield、61～499ブロックは DigiShield と LWMA を併用し、500ブロック目以降は zawy12 の LWMA のみが使用されます。
初期のマイニング報酬は1ブロックにつき50 susucoin で、945,000ブロック毎に半減します。
Segwit はブロック高0からサポートされます。
Susucoin の総発行量は1億500万 susucoin で、そのうちの10％は事前に発行され、 Susucoin の開発、マーケティングや成長のために使用されます。
Susucoin の公開前に合計で5ブロックがプレマインされ、精査されています。OP_RETURN は＜まとめ＞において説明される次なる目的を容易にできるように、のちのソフトフォークで512バイトに設定されます。
我々は独自のテキスト圧縮アルゴリズムと格納プロトコルの開発に取り組んでおり、詳細はのちに512バイトのOP_RETURN へのソフトフォークの前に発表する予定です。

まとめ
--------
Susucoin Core チームはすでに言論の自由を守る多数の匿名プラットフォームと、その記録能力を高めることを目的とした第一者提携を結んでいます。
要求に応じて、投稿されたテキストは Susucoin のブロックチェーンに記録され、半永久的に保存されます。
Susucoin Core チームは Susucoin のブロックチェーン上に保存されたデータを閲覧しやすくするために便利なブロックチェーンビューアを提供する予定です。
Susucoin のブロックチェーン上に書き込むことで、世界中のすべての人々の考えを改変されることなく半永久的に記録していくことが可能になります。
また Susucoin はある API を開発する予定です。その API とは、Susucoin Core チームと提携した言論の自由を守る匿名プラットフォームに実装される、便利なウェブインターフェースを通して、
匿名の利用者が互いに Susucoin をやりとりするのを可能にするものです。
Susucoin Core チームはより簡単に、思想や言論をブロックチェーン上に記録するために、新たなテキスト圧縮アルゴリズムと格納プロトコルの開発に取り組んでいます。
